# Walkthrough

1. Abra o vault `/Users/diogenes/vaults/usg_treinamento` no Obsidian.
2. Comece por `content/00-inicio.md`.
3. Edite a ordem ou o texto das páginas conforme sua aula.
4. Rode `python3 scripts/build_site.py`.
5. Abra `index.html` no navegador.
6. Durante a aula, use as páginas na ordem do menu lateral.
7. No final, direcione a equipe para `docs/checklists.md` e `content/06-armadilhas.md`.
