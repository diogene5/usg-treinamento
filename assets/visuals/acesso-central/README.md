# Imagens de acesso central

Imagens selecionadas como referência visual para o módulo de acesso central guiado por ultrassom.

## Origem

As imagens `saem-*.png` foram organizadas a partir de arquivos locais baixados da página:

- SAEM. Venous access. https://www.saem.org/about-saem/academies-interest-groups-affiliates2/cdem/for-students/online-education/m3-curriculum/bedside-ultrasonagraphy/venous-access

Créditos visíveis na página SAEM:

- `saem-anatomia-pescoco-ijv-cca.png`: imagem cortesia de Sierra Beck, MD.
- `saem-veia-arteria-parede.png`: imagem cortesia de Sierra Beck, MD e Bradley Wallace, MD.
- `saem-eixo-curto-armadilha-ponta.png`: imagem da seção de comparação de métodos da página SAEM.
- `saem-guia-fio-ijv.png`: imagem cortesia de Sierra Beck, MD e Bradley Wallace, MD.
